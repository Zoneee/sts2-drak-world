## 二、原版体系卡牌设计思路

### 2.1 铁甲战士（The Ironclad）

**定位**：入门友好的力量型战士，通过 HP 消耗换取强力收益；体系核心是 Strength（力量）积累与 Exhaust（消耗）协同。
**初始遗物**：Burning Blood——每场战斗结束后恢复少量 HP。

| 体系               | 核心循环                                                  | 代表关键词/机制                                                      |
| ------------------ | --------------------------------------------------------- | -------------------------------------------------------------------- |
| **力量强化流**     | 积累 Strength → 多段攻击指数倍增                          | Strength、Vulnerable（放大输出）、多段攻击（Twin Strike, Whirlwind） |
| **格挡转伤流**     | 高格挡堆叠 → Body Slam 将格挡转为伤害                     | Block、Barricade（格挡不消失）、Juggernaut（每次格挡造成伤害）       |
| **消耗协同流**     | 打出技能牌触发消耗 → Feel No Pain / Dark Embrace 被动收益 | Exhaust、Corruption（技能牌费用 0 且消耗）、多 Power 叠加            |
| **放血反伤流**     | 打出有 HP 代价的牌 → Rupture 将受伤转为 Strength          | 自我伤害、Rupture（受伤 = 力量）、Inferno（受伤 = AoE 伤害）         |
| **"Strike"攻击流** | 保留大量 Strike 类牌 → Perfected Strike 大伤害            | Strike 关键词计数、能量效率最大化                                    |

**设计规律**：铁甲战士以"用资源（HP、手牌）换节奏与爆发"为核心，Power 牌（持久效果）往往是体系转折点。上限较低但容错率高。

---

### 2.2 沉默者（The Silent）

**定位**：高操作上限的速攻型敏捷角色，通过大量抽弃牌和控制类减益削弱敌人；STS2 新增专属关键词 **Sly（奇巧）**。
**初始遗物**：Ring of the Snake——首回合额外抽 2 张。

| 体系               | 核心循环                                                                   | 代表关键词/机制                                                                        |
| ------------------ | -------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| **Shiv（匕首）流** | 生成海量 Shiv → Accuracy 双倍伤害 → 关键字触发类遗物                       | Shiv（0费攻击，打出后消耗）、Accuracy、Exhaust 协同遗物（Joss Paper 等）               |
| **毒流**           | 快速叠毒 → Accelerant 加速毒伤 → 持久压制                                  | Poison（中毒）、Noxious Fumes（被动持续叠毒）、Bubble Bubble（毒加倍）                 |
| **Sly（奇巧）流**  | 弃牌触发 Sly 牌免费打出 → Tools of the Trade / Master Planner 全面赋予 Sly | Sly（奇巧）、Tools of the Trade（每回合自动弃牌）、Tactician（弃牌 +1 能量）、高速循环 |

**设计规律**：沉默者以"薄牌组 + 高速循环 + 弃牌联动"为核心。Sly 体系是 STS2 新增的标志性机制，弃牌即打牌，形成独特的手牌节奏层。

> **与本 Mod 的交叉**：沉默者的 Sly（弃牌触发）与本 Mod 的弃牌触发机制高度相似，但规则不同——Sly 是"免费再打一次"，本 Mod 是"触发被弃牌自身的效果"。

### 2.3 死灵缚魂师（The Necrobinder）

**定位**：全新角色，以召唤骷髅坐骑 **Osty** 为核心资源的操控型角色。通过提升 Osty 的 HP 与调用 Osty 的攻能来创造输出与防御价值。
**初始遗物**：与 Osty 相关（具体数值待补充）。

| --------------- | ------------------------------------------------ | -------------------------------------- |
| **Osty 强化流** | 反复召唤提升 Osty 最大 HP → 利用 Osty 的攻击能力 | Summon（召唤）、HP 管理、Osty 出击指令 |
| **其他体系**    | 待补充                                           | —                                      |

### 2.4 摄政王（The Regent）

**定位**：全新角色，以 **Stars（星力）** 资源和 **Forge（锻造）** 武器为双核机制的爆发型角色；设计偏向高费用牌的"资源转化"。
**初始遗物**：Crown of Stars——每场战斗开始时获得 3 Stars。

| 体系                              | 核心循环                                                                   | 代表关键词/机制                                                            |
| --------------------------------- | -------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| **主权之刃（Sovereign Blade）流** | 持续 Forge 提升主权之刃伤害 → 用它作为主要终结手段                         | Forge（锻造）、Conqueror（双倍伤害）、高能量需求                           |
| **星力（Stars）流**               | 积累 Stars → 用 Stars 打出高消耗或零费强力牌 → Alignment 将 Stars 换为能量 | Stars（星力）、Genesis（星力引擎）、Gamma Blast / Falling Star（施加易伤） |

**设计规律**：摄政王的两个体系可以交叉融合——星力生成为锻造体系提供能量余量，主权之刃为星力体系提供终结能力。核心挑战是在"生成资源"和"使用资源"之间保持平衡。

> **与本 Mod 的关系**：本 Mod（STS2_Discard_Mod）的 10 张牌均属于 Regent 的卡牌池（`RegentCardPool`），提供了 **弃牌触发（Discard Synergy）** 这一第三条路径，与官方 Blade 和 Stars 体系并列，侧重链路操控与多时点价值兑现。

---

### 2.5 黑铁机（The Defect）

**定位**：STS1 回归角色，以 **Orb（法球）** 系统为核心的技术型角色；STS2 中法球系统升级并新增了 Glass Orb，Claw 流也得到加强回归。
**初始遗物**：Cracked Core——每场战斗开始时装填 1 个闪电法球。

| 体系               | 核心循环                                                   | 代表关键词/机制                                                                                                        |
| ------------------ | ---------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| **Claw（爪击）流** | 每次打出 Claw 类牌，所有 Claw 伤害 +2 → 0 费牌循环无限飞轮 | Exhaust 协同、0 费牌密集、All for One / Hologram 回收关键牌                                                            |
| **Orb 循环流**     | 装填并驱逐 Orb 触发效果 → Focus 放大 Orb 收益              | Channel（装填）、Evoke（驱逐）、Focus（专注）、各类 Orb 分工（Lightning 单体、Frost 格挡、Dark 蓄力 AoE、Plasma 能量） |

**Orb 类型分工**：
- **Lightning（闪电）**：驱逐时对随机敌人造成伤害
- **Frost（冰霜）**：获得格挡
- **Dark（黑暗）**：每回合蓄积伤害，驱逐时对最低 HP 敌人爆发
- **Glass（玻璃）**：对所有敌人造成伤害，但伤害逐回合递减
- **Plasma（等离子）**：获得能量

## 三、主流卡组设计规律

### 3.1 无限流（Infinite / Loop）

**定义**：通过抽牌 + 弃牌（或其他循环手段）在单回合内无限次循环牌库，打出海量卡牌造成压倒性伤害或防御。

**核心循环逻辑**：
`抽牌 → 打若干牌 → 弃牌 → 牌库触底重新洗入 → 继续抽牌` → 无限重复

**构筑要点**：
- 需要净正的"抽牌效率"（每次循环抽到的牌能支撑打出抽牌牌）
- 需要能量足够（Hellraiser、Pyre 等能量来源）
- 代表卡牌：铁甲的 Pommel Strike + Hellraiser；黑铁的 Claw + Hologram/All for One

**风险**：无限循环一旦被剪断（如 Smoggy），立刻缺乏过渡手段。

---

### 3.2 力量/属性强化流（Strength Scaling）

**定义**：通过持续积累 Strength（或其他攻击倍率来源），让有限的攻击卡造成指数级增长的伤害。

**核心循环逻辑**：
`早期 Power 牌积累力量 → 中期多段攻击放大收益 → 后期1~2张攻击牌可以秒杀BOSS`

**构筑要点**：
- 前期需要生存手段撑到 Power 牌生效（格挡/减益）
- 多段攻击 > 单段攻击（每段独立享受力量加成）
- 代表卡牌：Demon Form（每回合被动 +2 Strength）、Inflame、Barrier 类防御支撑

**标志性角色**：铁甲战士（Strength 是其一等体系）

---

### 3.3 毒/持续压制流（Poison / DoT）

**定义**：快速叠加足量中毒层数，靠自动扣血消灭 BOSS；战斗中期转入格挡/减益维持生存。

**核心循环逻辑**：
`前期：叠毒 → 中期：Accelerant 加速毒伤 / Burst 放大技能效果 → 后期：毒伤收束`

**构筑要点**：
- 对面有 Artifact 时毒无法施加，需要先消耗圣物
- 前 1~2 回合通常无输出，需要控制类减益或格挡撑住
- 单体 BOSS 效果极好；多体战斗（AOE 毒如 Noxious Fumes）效果依赖存活
- 代表卡牌：Deadly Poison、Poisoned Stab、Bubble Bubble、Accelerant（沉默者）

**标志性角色**：沉默者（Poison 是其核心体系之一）

---

### 3.4 控制流（Control / Debuff）

**定义**：通过持续施加虚弱（Weak）、易伤（Vulnerable）等减益状态+格挡，拖延战斗并压制敌方输出，让低伤害输出牌足以应对高输入。

**核心循环逻辑**：
`施加减益 → 我方格挡大幅提升（敌方输出减少）→ 我方大幅受易伤加成（伤害放大）→ 常规伤害即可击杀`

**构筑要点**：
- Weak（-25% 敌方攻击）和 Vulnerable（+50% 受到攻击伤害）是最通用减益
- 本 Mod 的 `CripplingManuscript`（崩坏手稿）是此流的重要扩展：弃牌时对全体施加群体双减益
- 适合与输出较低但生存稳的体系结合（如格挡转伤、毒流）

---

### 3.5 保留/手牌操控流（Retain / Hand Management）

**定义**：利用 Retain（保留）类关键词跨回合积累强力手牌，或通过手牌数量触发特定奖励，在某一回合集中爆发。

**核心循环逻辑**：
`多回合积累关键手牌 → 单回合集中打出 → Convergence / Runic Pyramid 等保持手牌`

**构筑要点**：
- 弱点：手持大量牌时，每回合能量不足以全打出
- 需要额外能量来源或减费手段
- Runic Pyramid（Ancient 祝福）：回合结束不弃牌，但会导致无法抽新牌——需要平衡
- 摄政王的 Convergence（整手牌 Retain，次回合退款）是该体系的代表牌

---

### 3.6 弃牌触发流（Discard Synergy）

**定义**：通过主动弃牌触发牌面上的"弃牌效果"，使同一张牌在打出时和被弃掉时都产生价值，形成抽弃链路的多时点收益。

**核心循环逻辑**：
`打出带弃牌效果的牌（打出效果）→ 弃掉手中其他带效果的牌（弃牌触发）→ 被弃掉的牌有时会再次引发弃牌 → 链路扩展`

**设计分层**：

| 层级                   | 角色               | 机制描述                                      |
| ---------------------- | ------------------ | --------------------------------------------- |
| **沉默者 Sly**         | 沉默者             | 弃牌 → 免费打出，适合攻击 / 格挡类 Sly 牌     |
| **本 Mod 弃牌体系**    | 摄政王（Mod 专属） | 弃牌 → 触发该牌自身的第二效果（特殊时点兑现） |
| **通用 Tactician**     | 沉默者             | 弃牌 → 获得能量（弃牌即产能）                 |
| **Tools of the Trade** | 沉默者             | 每回合开始自动弃牌 → 被动触发 Sly 类牌        |

**本 Mod（STS2_Discard_Mod）弃牌体系核心规则**：
- **默认**：只有因卡牌效果引发的弃牌才触发弃牌效果
- **例外**：`FadingFormula`（褪色公式）牌面明确可被回合末自弃触发
- 弃牌效果具有奇偶分支、群体 vs 单体切换、链路放大等设计
